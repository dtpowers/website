# Personal website

Static single-page portfolio. Replace placeholder content in `index.html` with your name, about text, accomplishments, and project links.

## Run locally

- **Option 1:** Open `index.html` in a browser (file://). Some features (e.g. fonts) may vary.
- **Option 2:** Serve the folder with a local server, e.g.:
  - `npx serve .`
  - `python -m http.server 8000` (then open http://localhost:8000)

## Deploy

Upload the contents of this folder to any static host (GitHub Pages, Netlify, Vercel, etc.), or connect the repo and use the root (or `website`) as the publish directory. No build step required.
